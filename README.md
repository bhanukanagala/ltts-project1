# ltts-project1
Here's your full Project/Task Tracking Dashboard.
✅ Features included

4 KPI cards — Total Tasks, Completed, Blocked, Overall Progress
Sprint Burndown line chart (remaining vs completed)
Sprint Velocity bar chart (story points per sprint)
Status Breakdown pie chart
Sortable data table — click any column header to sort
Filter tabs — All / In Progress / Completed / Blocked / Backlog
Live search by task name, project, or assignee
Progress bars with color coding (green = done, red = blocked)
Fully typed with TypeScript, responsive layout
